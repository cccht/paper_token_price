# Cover Letter Draft For Simulation Modelling Practice and Theory

Dear Editor,

Please consider our manuscript, **"Simulation-Based Time-of-Use Pricing for LLM
Inference Under Fixed Capacity: Provider Competition, QoS Protection, and the
Profit Boundary"**, for publication in *Simulation Modelling Practice and
Theory*.

The manuscript develops an auditable simulation model for a fixed-capacity LLM
inference-service market with two heterogeneous providers, one API intermediary,
time-rigid and time-flexible users, direct-provider access, and an outside
option. The model connects pricing, user choice, intermediary routing, load,
utilization, QoS degradation, and profit in one simulation workflow. It is
positioned for SMPT because the contribution is primarily a transparent
modelling and simulation study: the paper states the formal model, reports the
solution procedure, verifies fixed-point and solver behavior, and separates
validated evidence from credibility boundaries.

The main result is deliberately bounded. Under congestion, time-of-use pricing
reduces peak utilization and raises minimum QoS across the reported solver
objects. A finite-grid double-oracle mixed-strategy check on the full 225-point
grid reaches maximum regret 0.203. At the same time, the profit effect is not
robust: the sign changes across the coarse snapshot, fine snapshot, and
low-regret mixed profile. The manuscript therefore presents time-of-use pricing
as a QoS-protection instrument under fixed capacity, not as a reliable
profit-improvement mechanism.

We believe the paper is suitable for *Simulation Modelling Practice and Theory*
because it addresses a current AI-service capacity-management problem through a
reproducible simulation model, explicit verification and validation boundaries,
finite-grid solver diagnostics, and sensitivity/stress checks that make the
scope of the conclusions clear.
