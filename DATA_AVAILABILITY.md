# Data Availability Statement

The simulation code, model configuration, scripts, generated JSON/CSV artifacts,
figures, and LaTeX sources are available in the public GitHub repository:

<https://github.com/cccht/paper_token_price>

The main artifacts are available under repository paths such as:

- `artifacts/peak_shaving/20260618/`
- `artifacts/peak_shaving/20260619/`
- `artifacts/peak_shaving/20260619_submission/`
- `artifacts/peak_shaving/20260619_smpt/`
- `figures/peak_shaving_diagnostics/`
- `figures/peak_shaving_submission/`
- `figures/peak_shaving_smpt/`

For formal submission, the authors should archive a frozen repository release on
Zenodo, OSF, or another persistent repository and replace this statement with
the final DOI or permanent URL.

The public API price values are used only as scale anchors. The controlled vLLM
measurements are used only as QoS-shape anchors and do not constitute production
trace calibration.
