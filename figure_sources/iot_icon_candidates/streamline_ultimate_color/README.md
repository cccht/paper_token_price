# Streamline Ultimate Color Icon Candidates

This folder contains original, unmodified SVG assets selected for the paper's
market and simulation framework figure. The selection follows the visual style
of the supplied reference: dark outlines, light neutral bodies, and restrained
yellow, blue, red, and green fills.

The SVG bytes are copied directly from the pinned upstream commit. File names
are changed only to make their intended paper roles easier to identify.
`SOURCES.tsv` maps every local file to its exact upstream path.

## Scope

The 48 assets are divided into six groups:

- IoT endpoints
- Market actors
- API routing
- GPU capacity and QoS
- Pricing and time
- Simulation evidence

These are raw candidates only. No icon has been combined into a framework
figure, inserted into a TeX file, recoloured, cropped, or otherwise modified.
